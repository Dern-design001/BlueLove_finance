# 💎 Bluelove Studio | Finance & Inventory Dashboard

A premium, glassmorphism-themed web application for managing boutique finance and material inventory.

## ✨ Features
- **Finance Stream**: Real-time revenue and expense logging with category filtering.
- **Material Inventory**: Track stock quantities, color variants (HEX), and low-stock alerts.
- **Analysis**: Dynamic doughnut charts for financial health visualization.
- **Premium Design**: Glassmorphism cards, animated background orbs, drifting hearts, and twinkling sparkles.
- **Responsive**: Fully optimized for mobile and desktop views.
- **Local Persistence**: Data is saved to browsers' local storage for immediate access.

## 🛠️ Built With
- **Frontend**: HTML5, Vanilla JavaScript, CSS3
- **Styling**: Tailwind CSS, Custom Animations
- **Charting**: Chart.js
- **Fonts**: Plus Jakarta Sans (Google Fonts)

## 🚀 How to Run
1. Clone the repository.
2. Open `index.html` in any web browser.
3. Recommended: Use a live server (e.g., VS Code Live Server or `npx live-server`) for the best experience.
